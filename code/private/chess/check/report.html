$\color{red}\text{判断将军}$

判断各个棋子是否能控制到王的过程，在本题中起着重要的作用。

$\color{purple}\text{车}$

车可以控制同一行或同一列的棋子。使用$4$个$\text{for}$循环，遇到王就返回$\text{true}$，遇到其他棋子立刻跳出循环，以防止越子行为。

$\color{purple}\text{象}$

车可以控制与其同主对角线和同副对角线上的棋子。同样使用$4$个$\text{for}$循环，只不过有两个变量来表示坐标。

$\color{purple}\text{后}$

后的走法是车与象的集合，所以只要这两个走法有任意一种能控制到王，后就能控制到王。
